# 우리 가족 블로그 👨‍👩‍👧‍👦

스마트폰과 PC에서 바로바로 글을 쓰고 사진을 올릴 수 있는 가족 전용 블로그입니다.
관리자가 초대코드로 가족 구성원을 초대하고, 구성원은 로그인 후 바로 이야기를 남길 수 있어요.

## ✨ 주요 기능

### 프론트
- **모바일 퍼스트 UI** — 아이폰·안드로이드에서도 탭하기 쉬운 레이아웃
- **매거진 스타일 피드** — 큰 썸네일, 카테고리 칩 필터, 검색
- **게시글 상세 페이지** — 사진 갤러리, 좋아요, 댓글, 라이트박스
- **빠른 글쓰기** — 다중 사진 업로드, 드래그&드롭, 붙여넣기, 대표 이미지 선택
- **갤러리 페이지** — 전체 사진 타일 모음
- **구성원 페이지** — 누가 얼마나 글을 썼는지 한눈에

### 관리자 (admin 역할만 접근)
- **대시보드** — 구성원/게시글/사진/댓글 통계, 최근 글, 작성자 랭킹
- **구성원 관리** — 직접 추가/수정/삭제, 역할 변경, 비활성화
- **초대코드 발급** — 역할/만료일 지정, 초대 링크 자동 복사
- **게시글 관리** — 전체 글 목록, 수정/삭제

### 인증 & 보안
- JWT 쿠키 기반 로그인 (30일 유지)
- bcrypt 비밀번호 해시
- 초대코드가 있어야만 회원가입 가능 (공개 가입 불가)

## 🚀 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 초기 관리자 계정과 샘플 데이터 생성
npm run seed

# 3. 서버 시작
npm start
```

브라우저에서 http://localhost:3000 접속

### 초기 계정
| 역할 | 아이디 | 비밀번호 |
|------|--------|----------|
| 관리자 | `admin` | `admin1234` |
| 멤버 | `mom` | `member1234` |

**로그인 후 반드시 `/me` 에서 비밀번호를 변경해 주세요.**

## 🧭 페이지 안내

| URL | 설명 |
|-----|------|
| `/` | 블로그 메인 피드 |
| `/post/:id` | 게시글 상세 |
| `/write` | 새 글 / 수정 (로그인 필요) |
| `/gallery` | 사진 전체 보기 |
| `/members` | 가족 구성원 |
| `/me` | 내 프로필 / 비밀번호 변경 |
| `/admin` | 관리자 패널 (admin 전용) |
| `/login`, `/register` | 로그인 / 가입 |

## 👨‍👩‍👧 새 가족 구성원 초대하기

1. 관리자로 로그인 → `/admin` → **초대코드** 탭
2. **초대코드 생성** → 역할/만료 선택
3. 발급된 링크를 가족에게 카카오톡 등으로 공유
4. 가족이 링크를 열어 `/register`에서 아이디/이름/비밀번호 입력

## 🛠 기술 스택
- **런타임**: Node.js 22+ (`node:sqlite` 내장 모듈 사용)
- **서버**: Express + JWT + bcryptjs
- **업로드**: Multer (`/uploads` 로 파일 저장)
- **프론트**: Vanilla JS ES modules + 모던 CSS (빌드 과정 불필요)
- **DB**: SQLite 단일 파일 (`data/blog.db`)

## 📂 구조
```
family/
├── server.js          # 메인 서버
├── db.js              # DB 스키마 + 헬퍼
├── seed.js            # 초기 데이터
├── routes/            # API (auth, posts, users, uploads)
├── middleware/        # 인증 미들웨어
├── public/            # 정적 프론트 (HTML/CSS/JS)
├── data/blog.db       # SQLite (자동 생성)
└── uploads/           # 업로드된 이미지
```
